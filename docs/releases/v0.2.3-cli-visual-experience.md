# Boronix v0.2.3 CLI Visual Experience

Release notes for `v0.2.3`:

- **Compact Dev Startup screen**: Refactored `boronix dev` and `boronix start` to print clean, minimalistic boot information.
- **ASCII tree branch rails**: Replaced basic lists with structured tree layouts using Unicode rail glyphs (`│`, `├─`, `└─`) and fallback characters.
- **Live server request log**: Implemented real-time request audits mapping to `serve`/`render`/`api`/`action`/`miss`/`error` kind definitions.
- **New commands**: Added `boronix routes` and `boronix inspect` utilities.
- **LAN network URL**: Resolved local LAN IPv4 addresses dynamically.
- **Auto browser opening**: Added support for `--open` options in development.
- **Actionable error formatting**: Integrated hints and expected/found output structures for port clashes, invalid loaders, and missing action endpoints.
