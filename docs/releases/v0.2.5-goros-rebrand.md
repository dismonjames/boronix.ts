# Goros v0.2.5 Release Notes

This milestone release rebrands the entire framework from **Kumquat.ts** to **Goros** and prepares package tarballs for npm release.

## Brand Modifications

- **Core Package**: Rebranded from `kumquat` to `goros`.
- **Creator Package**: Rebranded from `create-kumquat` to `create-goros`.
- **CLI Utility**: Rebranded binary and executions from `kumquat` to `goros`.
- **User Imports**: All framework features are now imported via `goros` instead of `kumquat`.
- **Configuration File**: Changed the primary workspace configuration filename to `goros.config.ts`.
- **Build Output**: Re-routed production manifest compilations to `.goros/` instead of `.kumquat/`.

## Deprecation Warnings

- If `kumquat.config.ts` is detected in the root directory, Goros will print a deprecation warning guiding users to rename it to `goros.config.ts`.
- If an old `.kumquat/` manifest is found, Goros will fail execution guiding users to rebuild the manifest under `.goros/`.
