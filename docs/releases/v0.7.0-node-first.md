# v0.7.0 - Node-first Runtime

Boronix is now a Node-first framework. Node.js is the primary and default runtime, with Bun remaining as an optional compatibility runtime.

- **CLI shebangs**: Updated to run through Node.js.
- **Package Manager**: npm is the canonical package manager.
- **Production Build**: A self-contained compiled JS bundle is emitted, running directly without tsx or Bun.
- **SQLite Database**: Replaced Bun-specific SQLite driver with `@libsql/client` and `drizzle-orm/libsql`.
- **Node support floor**: >= 18.18.

See [the migration guide](../migration-v0.7.0.md) for how to migrate from previous versions.
