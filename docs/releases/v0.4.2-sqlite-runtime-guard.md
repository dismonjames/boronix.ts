# Boronix v0.4.2 - SQLite Runtime Guard

Boronix v0.4.2 tightens the database scaffolding contract.

Included:

- `create-boronix --db sqlite --runtime node` is rejected before files are created.
- SQLite documentation now clearly requires Bun runtime.
- Postgres documentation now clearly supports Bun and Node.

No database engine or ORM logic changed.
